# DSA-Marketing-Project
The purpose of this project is to present a business intelligence report on the recent sales performance of DSA Marketing Ltd. It delivers timely, precise and user-friendly information, facilitating rational choices. The dashboard displays a holistic view of KPIs and financial indicators and assists in finding opportunities and challenges.
>This project showcases the usage of Power Query and M Code inside the Power BI software. Multiple tables were also linked and connected within the software and used to create a dashboard to visualize the data in hand.
